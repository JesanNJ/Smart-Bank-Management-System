# Smart-Driven Bank Management System

An advanced, console-based C++ banking application that integrates core financial management tools with a **Smart Analyzer**. It allows users to track their spending habits, set monthly budgets, and define long-term savings goals while dynamically assessing their financial health.

## Key Features
* **Financial Insights Chatbot (Gemini 3):** A dedicated chatbot window that leverages a **Python-based AI Bridge** to send your questions to the **Gemini 3 Flash Preview API** (Google Generative AI). Replies are generated by Gemini 3 using your account's financial data. Set the `GEMINI_API_KEY` environment variable or update the `.env` file to use it.
* **Smart Financial Analyzer:** Automatically analyzes spending behavior across different categories, evaluates the user's savings ratio, and scores overall financial health (0-100).
* **Budget Tracking:** Set category-based and monthly limits to receive early warnings when risking overspending or living paycheck-to-paycheck.
* **Goal Simulation:** Track custom savings goals (e.g., "Emergency Fund", "Vacation") mapped against monthly progress and timeline estimates.
* **Automated Reporting:** Generates a real-time spending trend analysis (using percentage shares) and simplified monthly income/expense flows.
* **Granular Transaction Logging:** Stores a timestamped, highly granular breakdown of money-movement, mapping every withdrawal to 11 specific categories.

## Core Technologies
* **C++17**: High-performance core banking logic and user management.
* **Python**: Flexible AI bridge for interacting with LLM APIs (requires Python 3.x).
* **CMake**: Cross-platform build system.

## 🚀 Getting Started

### 1. Build the C++ Application
To compile the application using CMake:
```bash
mkdir build
cd build
cmake ..
cmake --build .
```

### 2. Configure the AI Chatbot (Grok)
The chatbot requires an xAI API key and Python 3.

1.  **Get an API Key**: Visit the [xAI Console](https://console.x.ai/team/default/api-keys).
2.  **Set the API Key**:
    *   **Option A (.env)**: Create/Update the `.env` file in the root directory:
        ```env
        XAI_API_KEY=your_xai_api_key_here
        ```
    *   **Option B (Environment Variable)**:
        *   Windows: `$env:XAI_API_KEY = "your_key"`
        *   Linux/macOS: `export XAI_API_KEY="your_key"`

### 3. Run the System
Following the build step, run the executable binary:
```bash
./build/bank_management
```
Select **Option 9** from the main menu to start the **Financial Insights Chatbot**.
